# Political Affiliation Detection on the Archelec Corpus

ML FOR NLP — Rafaël Mourouvin

## Contents

- `ML_FOR_NLP.pdf` — written report 
- `ML_FOR_NLP.ipynb` — Jupyter notebook with the full pipeline: data loading, EDA, classification (RQ1), LDA semantic mapping (RQ2), and temporal evaluation (RQ3)

## Data

The text files are not included in this repository.
- Text files: clone from https://gitlab.teklia.com/ckermorvant/arkindex_archelec (done in the notebook)
- Metadata CSV: use archelect_search_73_93.csv

## How to run

Open `ML_FOR_NLP.ipynb` in Google Colab. The notebook installs all dependencies and clones the text repository automatically. Upload the metadata CSV manually before running the data loading cells.
